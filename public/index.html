About

This API is built with Node using the Express framework. It scrapes Google Finance for data and returns it to the user in the form of JSON.

API Calls

Last quote for a given symbol: /api/quote/*

Example:

/api/quote/GOOG

/api/quote/TSE:BB

/api/quote/amazon

Usage

Requests to the demo server (i.e this one) are limited to one every 60 seconds.
Also, please don't use this for your trading - this API is basically a toy and shouldn't be used for investing purposes.
